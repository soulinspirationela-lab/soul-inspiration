# Example content for Soul Inspiration

This is a simple Markdown example you can use as a template for notes, blog posts, or documentation.

## Tips
- Put images in `assets/images/` and reference them as `/assets/images/your-image.png`.
- Use the shared stylesheet `assets/css/styles.css` for consistent styling.

## Example
Write your content here, using Markdown. When you need an HTML page, use `examples/sample.html` as a starting point.
